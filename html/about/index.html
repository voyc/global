About voyc

voyc is a software development group specializing in software systems for learning. All projects are open source and hosted publicly on GitHub.

Applications

  • voyc
    Search engine with travel UI.
  • plunder
    Global history treasure hunt game.
  • guru
    Display a random inspiring quote.
  • sanskrit
    Sanskrit keyboard. For typing or learning.
  • flash
    Smart Flash Card system that knows you and feeds you what you need.
  • pokerface
    Algorithmetically derived emotional facial expressions.
  • vote
    Demonstrate Instant Runoff Voting.
  • curriculum
    Everything a person needs to learn to do AI.

Submodules

  • jslib
    Library of javascript utility functions.
  • minimal
    Minimal styles and behaviors for a website.
  • icon
    A collection of icons - character or canvas.
  • account
    User management: login, logout, etc.
  • model
    Application template, including navigation, event-driven flow, user management, and all the features of the other submodules.
